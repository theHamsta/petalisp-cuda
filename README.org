* petalisp-cuda

** Usage

#+BEGIN_SRC lisp
(quicklisp:quickload :petalisp-cuda)
(petalisp-cuda:use-cuda-backend)
#+END_SRC

** Requirements

- CUDA toolkit
- CUDNN

** Installation

With [[https://www.quicklisp.org/beta/][quicklisp]] installed, clone this
repository to your local projects folder:

#+BEGIN_SRC bash
git clone 
#+END_SRC
